# Millennium Asteroids

A 2D space shooter developed using the Godot Engine and C#. In this game, players pilot the Millennium Falcon, defending against relentless waves of Tie Fighters while trying to get a high score.

**Note:** This game is a semestral project developed for the Programming 2 course.
